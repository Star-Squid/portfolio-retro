How to play

  • Click "New Game" to begin
  • Click two of the game tiles to reveal either flowers or leaves, try to memorize their position
  • Choosing two tiles per round, try to match the correct leaf and flower
  • Reveal a matching pair and it will not be hidden again - you are one step closer to finishing
  • Uncovering all six matching pairs ends the game and stops the timer. Restart by clicking "New Game"

About

It uses Bootstrap, JavaScript and jQuery; there are some css and jQuery animation effects.
It also features original art by the author.
